# RDP-Portal 缺少的功能清单

说明：这是针对当前项目作为轻量级远程桌面管理工具（RDP 管理器）所建议的缺失功能和改进点，按优先级与类别组织，便于后续实现与拆分。

高优先级（建议优先实现）
- 配置/资料管理
  [x] 导入/导出配置（JSON/RDP），便于备份与迁移
  [x] 配置分组与标签（Group/Folder/Tags）以组织大量条目
- 连接稳定性与选项
  - 目标：保存并编辑完整的 `.rdp` 参数，支持 RD Gateway，并提供可配置的重连策略与诊断选项。
  - 任务（可拆分为多个 PR）：
    - [ ] 在 UI 中增加“高级 RDP 参数”编辑区域（原始 `.rdp` 文本或键值对视图），支持导入/导出 `.rdp` 文件并预览生效参数。
    - [ ] 将这些高级参数纳入导出/导入 JSON schema，使配置文件能完整包含 `.rdp` 原始字段。
    - [ ] 支持 RD Gateway 设置：网关地址、端口、网关凭据使用方式（独立/复用本地凭据）、是否启用网关证书验证。
    - [ ] 实现可配置的重连策略：重试次数、重试间隔（秒）、是否在后台自动重连、失败后提示或静默记录。
    - [ ] 在连接失败或重连过程中记录本地诊断日志（可选开启），并在 UI 中提供查看或导出日志的入口。
    - [ ] 为敏感项（凭据）使用安全存储（建议：Windows Credential Manager 或 DPAPI 加密），并在文档中说明实现方式。
  - 受影响文件（建议参考）：
    - `src/RdpFile.cs`：序列化/反序列化 `.rdp` 字段至 JSON
    - `src/MainForm.cs`、`src/MainForm.Designer.cs`：UI 增加高级参数编辑与重连设置
    - `src/Profile.cs`：将高级参数与网关设置并入配置模型
  - 验收标准：
    - 能通过 UI 保存并导出包含完整 `.rdp` 参数的 JSON 配置。
    - 能通过 UI 配置并使用 RD Gateway 连接（手工测试用例描述）。
    - 在连接中断时，按配置的重连策略触发重试并在 UI 中显示当前状态或错误信息。
    - 日志功能能导出用于故障诊断，且凭据未明文写入导出文件。
- UI/可用性
  - 右侧面板使用响应式布局（TableLayoutPanel）以保证各宽度下对齐一致
  - 托盘图标与最小化到托盘选项（在后台保持运行）

中优先级（功能增强）
- 安全与隐私
  - 数据库/文件加密（本地配置文件加密，支持主密码）
  - 可选的匿名使用日志或本地日志（便于故障诊断）
- 集成与自动化
  - 命令行接口（CLI）以便脚本化连接与导入导出
  - 支持从 AD / LDAP 批量导入用户或主机（企业场景）
- 体验
  - 收藏/常用列表、连接历史与最近连接
  - 拖放排序与上下文菜单（右键：复制 IP、导出、打开 RDP 文件等）
  - 快捷键（Connect, New, Delete 等）

低优先级（可选、高级特性）
- 多会话 / 窗口管理
  - 在一个窗口中同时打开多个远程会话（选项卡或嵌入式视图）
- 远程屏幕预览
  - 在列表中显示缩略图或状态（在线/离线、分辨率信息）
- 同步/共享
  - 支持与云或网络位置同步配置（可选，需加密）
- 高级 RDP 控制
  - 证书管理（受信任证书、跳过证书警告选项）
  - 性能调优开关（带宽限制、编码选项）

工程质量与发布
- 打包与分发
  - 生成安装包（MSI 或自解压安装器），并保持签名（已有签名配置）
- 文档
  - README 补充使用手册：配置说明、常见问题、打包/发布说明

5. 添加托盘图标和“最小化到托盘”选项。
